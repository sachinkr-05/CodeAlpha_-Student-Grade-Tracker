# 🎓 Student Grade Tracker

The **Student Grade Tracker** is a Java-based application developed to efficiently manage and analyze student academic performance. This project provides a structured way to store, process, and evaluate student grades using fundamental programming concepts.

It enables users to input student data, compute key performance metrics, and generate a comprehensive summary report. The application is designed with simplicity and clarity in mind, making it suitable for both beginners and intermediate Java learners.

---

## 🚀 Features

* 📥 Add and manage student records
* 📊 Calculate average, highest, and lowest scores
* 🧮 Efficient data handling using Arrays or ArrayLists
* 📄 Generate detailed performance summary reports
* 💻 Console-based or GUI-based implementation

---

## 🛠️ Technologies Used

* Java
* Object-Oriented Programming (OOP)
* Data Structures (Array / ArrayList)

---

## 📂 Project Structure

* `Main.java` → Entry point of the program
* `Student.java` → Student data model (if OOP used)
* `GradeManager.java` → Logic for managing grades and calculations

---

## 📈 How It Works

1. User inputs student names and grades
2. Data is stored using Arrays or ArrayLists
3. System processes the data to calculate:

   * Average score
   * Highest score
   * Lowest score
4. A summary report is displayed to the user
